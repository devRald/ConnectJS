Getting Started

Welcome

Do you want your website respond with connection changes?

With the help of ConnectJS you can handle internet connection events and decide whether to use local storage strategies or stick with internet functionalities.

Features

  • Handle events online and offline
  • Check Internet Connection
  • Easy install; 100% integration

isConnected

  • $$.isConnected
    • return type - boolean
<script>
        $$.connected(function(event){
          //Type code here.
        })
</script>

Connected

  • $$.connected( Function : fn )
    • fn - callback function to handle when connected to the internet.
<script>
      	$$.connected(function(event){
          //Type code here.
        })
</script>

Disconnected

  • $$.disconnected( Function : fn )
    • fn - callback function to handle when disconnected to the internet.
<script>
        $$.disconnected(function(event){
          //Type code here.
        })
</script>

Status Change

  • $$.statusChange( Object : obj )
    • obj - object that handles both onConnect and onDisconnect
<script>
        $$.statusChange({
          onConnect:function(){
            //Type code here
          },
          onDisconnect:function(){
            //Type code here
          }
        })
</script>
